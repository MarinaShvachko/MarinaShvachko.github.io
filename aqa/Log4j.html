Log4j for loggin in Java

This is a logging framework written in Java with help of which logs can be printed to console or saved in files.

Library you need:

  
< dependency>
  < groupId>log4j< /groupId>
  < artifactId>log4j< /artifactId>
  < version>1.2.17< /version>
< /dependency>

Write logs to console:

Create a file with a name "log4j.properties" in a package: src → main → resources. Example of content:

log4j.rootLogger=INFO, stdout
log4j.appender.stdout=org.apache.log4j.ConsoleAppender
log4j.appender.stdout.Target=System.out
log4j.appender.stdout.layout=org.apache.log4j.PatternLayout
log4j.appender.stdout.layout.ConversionPattern=%d{yy/MM/dd HH:mm:ss} %p %c{2}: %m%n

Then in a class you want to write logs create a static variable with a type Logger and put the class name in getLogger() method. If there is a base class from that all other classes extend, it's better to create the variable in the base class. Thus all subclasses will have an access to it and you don't need to dublicate code.

public class HomePage {
  static final Logger log = Logger.getLogger(HomePage.class);
}

Now just use log object to call logging methods log.info("message from logger"). In console you will see 22/07/25 12:01:24 INFO HomePage: message from logger
