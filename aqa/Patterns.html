Page elements

POM

Page factory is an extension of pom

Annotations:

@FindBy - one criterion

@FindBys - some criteria and if one of them is met - an element is found (one any condition should be = true)

@FindAll - some criteria and only if all of them are met - an element is found (all conditions should be = true)

Elements that has annotation @Find(By\s\All) have to be initialized. It is done with a help of static method that we call in a construstor of pages

  
    public class HomePage extends BasePage {
    WebDriver driver;


    @FindBy(name = "uid")
    WebElement userIdField;

    public HomePage(WebDriver driver) {
        this.driver = driver;
        PageFactory.initElements(driver, this);
    }
