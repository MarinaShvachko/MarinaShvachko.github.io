Simple example how to get data from a response and put it in an another request

  1. Create a Collection in Postman
  2. Create request, the type of the request is Get
  3. Add URL https://httpbin.org/uuid
  4. Open Test tab:
  5. On the right side click "Set a global variable". In the Test tab appeared a piece of code. Change key and value data, shold be pm.globals.set("uuid", response.uuid); Add a variable let response = pm.response.json() This variable catch response.
  6. Create new request with a type Post
  7. Add URL https://httpbin.org/post
  8. Add Body, for example { "name": "Marina", "id": "{{uuid}}" here will be data from your variable }
  9. Run Get method and check the number uuid in response
  10. Then run Post method and check data in response. You will see that there is in response uuid that you sent in request. And this uuid you get from Get response

Automatically check status code:

Open Post request, open a tab Test. On the right find and click "Status code: Successful POST request"

The code was added in the Test tab. You can change the name of this method, for example pm.test("Verify response code", function () { pm.response.to.have.status(200); });

Run the Post method

Open a Tab "Test Results". There you can see that status code of the respons was checked automatically. You can change 200 on another number, run this Post request again and will see that there will be failed result.

Also you can check if a respons contains a parameter and asserts data in it. For example in our case we can check that in response there is the same uuid as in request:

  1. Open Post request
  2. Open Test tab
  3. On the right chose "Response body: JSON value check"
  4. The code in the Test tab will be added
  5. Add the path to a variable from a response and expected resultat
  6. Run Post method
  7. Open a Tab "Test Results". There will be 2 test now, one for checking status code and second checking name
